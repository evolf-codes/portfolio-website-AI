- 3. front end automation example

- 4. back end automation example

About:

Contact:

## Technical Details

- Implemented as a modern JS site, client rendered
- The JS app should be created in a subdirectory `frontend`
- No persistence
- No user management for the MVP
- Use popular libraries
- As simple as possible but with an elegant UI
